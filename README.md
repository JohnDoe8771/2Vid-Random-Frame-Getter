# Two-Video Random Frame Getter
A script to randomly extract two aligned frames from videos with the ability to automatically determine an offset if the videos' frames are misaligned.

```
Usage: ./aframes.sh [OPTIONS...] FILE_A FILE_B

Options:
  -c COUNT 	   Set the number of frames to extract
  -o OUTDIR	   Set the output directory
  -t TMPDIR	   Set temporary directory
  -a  		     Check the two neighbouring frames from B to see if they better match A on final extraction
```
